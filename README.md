# Sleep Tracker Analyzer

Анализ сессий сна из файла логов.

### Запуск

Через аргумент командной строки:

**java SleepTrackerApp путь/к/файлу.txt**

Если аргумент не указан, используется **sleep_log.txt** в корне проекта.

### Формат файла

#### ДД.ММ.ГГ ЧЧ:ММ;ДД.ММ.ГГ ЧЧ:ММ;КАЧЕСТВО

Качество указывается: GOOD, NORMAL или BAD

### Результаты

- Количество сессий сна
- Минимальная продолжительность
- Максимальная продолжительность
- Средняя продолжительность
- Количество сессий с плохим качеством
- Количество бессонных ночей
- Хронотип пользователя (Сова/Жаворонок/Голубь)
